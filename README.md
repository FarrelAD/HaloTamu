# HaloTamu 👋🏻

Website sederhana guna menunjang kebutuhan administrasi tamu di lingkungan kerja. Anda bisa memasangkannya pada perangkat seperti tablet atau komputer yang dipajang di bagian depan untuk para tamu sebelum masuk ke dalam acara.

# Kustomisasi 🎨

Anda bisa menggunakan aplikasi web sederhana ini pada perangkat anda dan bisa anda sesuaikan lagi dengan kebutuhan anda. Sebelum melakukan kustomisasi, anda perlu mengunduh repositori ini ke perangkat anda. Tata cara downloadnya sebagai berikut:
1. Klik tombol biru atau hijau bertuliskan `Code` di website ini 
2. Kemudian pilih opsi download ZIP

Untuk konfigurasi awal, anda bisa mengubahnya pada file `js/config.js`. Pada file tersebut terdapat beberapa konfigurasi seperti nama acara, data tamu, tema aplikasi, jadwal, dan juga lokasi acara. Untuk konfigurasi yang lebih kompleks, anda bisa mengubahnya pada file `js/app.js` ataupun juga bisa pada file `index.html`.
